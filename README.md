# 📚 Library Book Tracker

Library Book Tracker is a Java-based application designed to manage books in a library system. The system provides authentication, role selection, and a user interface for managing library operations.



## 🚀 Features

- User Login & Registration  
- Role Selection (User / Librarian)  
- Add, Borrow, Return, Delete Books  
- Search and Filter Books  
- GUI using Java Swing  


## 🧩 Project Structure

```
src/com/library/frontend/
├── AuthChoice.java
├── LoginUI.java
├── RegisterUI.java
├── RoleSelector.java (Main Entry Point)
├── LibraryUI.java
└── LibraryBackend.java
```


## 🛠️ Technologies Used

- Java  
- Swing  
- html
- JDBC  


## ▶️ How to Run

1. Compile all Java files  
2. Run `RoleSelector.java`
